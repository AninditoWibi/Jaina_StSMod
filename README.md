# Jaina_StSMod
基于《杀戮尖塔》(*Slay the Spire*)制作的炉石传说-法师吉安娜 Mod

### v0.2.0
实现了75张卡牌及衍生卡，在卡牌类型处显示了法术类型。

### v0.1.9
实现了60张卡牌及衍生卡，14种能力、5种伤害类型、4个卡牌标签、14个actions，人物立绘动画和自定义UI基本完成。

### v0.1.5
实现了冰冻效果、法术伤害能力；添加了暴风雪动画、人物立绘；添加了两张新卡；
